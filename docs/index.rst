Welcome to uarray's documentation!
==================================

`python-moa <https://github.com/costrouc/python-moa>`_ is an ambitious
project to provide a high level array operation interface that can
compile down into low level machine code. In addition moa can
provide predictions on the runtime and performance of a computation.
